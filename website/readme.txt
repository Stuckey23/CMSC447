How to setup VSCODE
Install Python https://www.python.org/downloads/
Install VSCODE https://code.visualstudio.com/download
Open the folder in VSCODE
open VSCODE terminal(CTRL + `)
pip3 install flask, flask_wtf, werkzeug, wtforms, numpy
go to command pallet (CTRL+SHIT+P)
search for Python Select: Interpretor
pick global
make sure the media file is empty before running the website or it will cause a index error
run app.py to start the website
http://127.0.0.1:5000
